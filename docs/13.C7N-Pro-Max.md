# 13. C7N Pro Max SKILL

用于为 AI 编程助手安装 C7N Pro Max 技能（Skill）的命令行工具。

## 安装

```bash
npm install -g @apaas-front/c7npro-cli --registry=http://nexus.saas.hand-china.com/repository/hzero-npm-group/
```

## 使用方法

```bash
# 为特定 AI 助手安装技能
c7npro init --ai claude      # Claude Code
c7npro init --ai cursor      # Cursor
c7npro init --ai windsurf    # Windsurf
c7npro init --ai antigravity # Antigravity
c7npro init --ai copilot     # GitHub Copilot
c7npro init --ai kiro        # Kiro
c7npro init --ai codex       # Codex (Skills)
c7npro init --ai roocode     # Roo Code
c7npro init --ai qoder       # Qoder
c7npro init --ai gemini      # Gemini CLI
c7npro init --ai trae        # Trae
c7npro init --ai opencode    # OpenCode
c7npro init --ai continue    # Continue (Skills)
c7npro init --ai all         # 为所有助手安装

# 选项
c7npro init --global         # 全局安装到用户主目录 ~/

# 卸载
c7npro uninstall --ai claude # 卸载 Claude Code 的技能
c7npro uninstall --global    # 卸载全局安装的技能
```

## 工作原理

本 CLI 仅使用本地资源进行安装，不依赖网络下载。安装时根据选择的 AI 平台类型，从内置模板生成相应的技能文件。
