11.5. max-lines

限制 React 组件代码行数

在 2.6. 代码行数规范章节中为了保证代码的可读性和后续可维护性，我们提出严格限制组件代码行数以避免过长的文件。

module.exports = {
  "rules": {
    "hand-apaas/max-lines": "warn"
  }
};

例子​

  • ❌错误
  • ✅正确
import React from 'react';

export default function MyApp() {
  // ... 此处省略 1000字
  return (
    <div>
      <h1>Welcome to my app</h1>
      <MyButton />
    </div>
  );
}
import React from 'react';

export default function MyApp() {
  return (
    <div>
      <h1>Welcome to my app</h1>
  <MyButton />
  </div>
);
