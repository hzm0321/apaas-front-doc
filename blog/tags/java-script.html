回调地域问题​

传统的异步编程方式通过回调函数，处理异步任务到期的执行问题。而异步任务之间如果产生依赖关系，就会陷入“回调地域”问题。

状态​

Promise 的状态不可逆
有3个状态

  • Pending
  • Resolved
  • Rejected

静态方法​

  • Promise.resolve
  • Promise.reject
