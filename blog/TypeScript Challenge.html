简单​

4. Pick​

实现一个 Pick<T, K>，从 T 中挑选出一系列 K 的属性。

例如：

interface Todo {
  title: string
  description: string
  completed: boolean
}

type TodoPreview = MyPick<Todo, 'title' | 'completed'>

const todo: TodoPreview = {
    title: 'Clean room',
    completed: false,
}
题解
type MyPick<T, K extends keyof T> = {
  [P in K]: T[P]
}
